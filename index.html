Palette Settings

What is dynamic chroma?

When enabled the base chroma gets determined automatically and individually for each hue. Every color uses its biggest possible chroma value (at the lightness value set at "Max chroma at lightness").

When enabled "Static chroma value" has no effect. Using this is usually not recommended since it gives all hues an uneven saturation.

Dynamic Hue Shift

What is bright colors mix hue?

Brighter colors (lightness > 0.5) get shifted towards this hue. The brighter the color the stronger the shift towards this hue. For warmer shifts this value should be around "80".

What is dark colors mix hue?

Darker colors (lightness < 0.5) get shifted towards this hue. The darker the color the stronger the shift towards this hue. For colder shifts this value should be around "260".

Show Base Color Values

What are base color values?

These values represent the lightness, chroma, and hue values of the base color. That's the color before Dynamic Hue Shift gets applied.

Currently, final colors get generated using the CSS function color-mix(). AFAIK I cannot extract those values from colors generated using that function.
